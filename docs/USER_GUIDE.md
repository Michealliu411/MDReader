# MD Reader 使用说明

MD Reader 是一个 macOS 上的轻量级 Markdown 阅读器,适合阅读 spec 文档、技术笔记、项目文档。

---

## 安装

### 从 DMG 安装

1. 双击 `MDReader-v3.dmg` 挂载磁盘
2. 把 **MD Reader** 拖入 **Applications** 文件夹
3. 打开 **MD Reader**

### 首次打开提示"已损坏 / 无法验证开发者"

这是 macOS 的安全机制(应用未做 Apple 代码签名)。解决方法:

**方法一(推荐):** 打开"终端",粘贴执行:
```bash
xattr -cr "/Applications/MD Reader.app"
```

**方法二:** 右键点 MD Reader → 选"打开" → 再点"打开"

> 之后就能正常打开了,只需操作一次。

## 基本使用

### 打开文档

| 方式 | 操作 |
|------|------|
| 打开单个文件 | 点工具栏 📄,或直接把 .md 文件拖进窗口 |
| 打开文件夹 | 点工具栏 📂,选一个文件夹,侧边栏显示树形目录 |
| 打开网络文档 | 点工具栏 🌐,粘 URL(支持 GitHub 链接) |

### 工具栏说明

```
📄 打开文件  │  📂 打开文件夹  │  🌐 网络文档  │  📤 导出HTML  │  A− 16 A+  │  🌙/☀️ 主题
```

| 按钮 | 功能 |
|------|------|
| 📄 | 打开本地 .md 文件 |
| 📂 | 打开文件夹(侧边栏显示文件树) |
| 🌐 | 输入 URL 拉取网络 md(GitHub 链接自动转换) |
| 📤 | 导出当前文档为 HTML 文件(有文档时才显示) |
| A− / A+ | 调节字号(12–24px) |
| 🌙 / ☀️ | 切换日间 / 夜间模式 |

### 阅读进度

- 自动记忆:读到哪个标题,关闭重开自动定位
- 进度按标题粒度(H1–H4),不需要手动保存

### 搜索

- 按 `⌘F` 唤出搜索栏
- 输入关键词,自动高亮匹配
- 点 ↑/↓ 跳转上一个 / 下一个匹配
- 点 `aa`/`Aa` 切换大小写敏感
- 按 `Esc` 关闭搜索

---

## 文件夹浏览

1. 点 📂 选一个文件夹
2. 侧边栏显示树形结构:
   - 📁 文件夹可点击折叠 / 展开
   - 📄 md 文件点击即打开
3. **自动记住**上次打开的文件夹,下次启动自动恢复

### 文件监听

打开的文件如果被外部编辑器(如 VS Code、文本编辑器)修改保存,**MD Reader 会自动刷新内容**,无需手动重开。刷新时保留你的滚动位置。

---

## 网络文档

### 支持的 URL

- **GitHub 文档**:直接粘 GitHub 链接,自动转为 raw 地址拉取
  - 例:`https://github.com/user/repo/blob/main/README.md`
- **任意 URL**:只要返回的是 Markdown 文本即可

### 缓存

- 拉取的文档会**缓存到本地**(7 天有效)
- 同一 URL 第二次打开**秒开**(用缓存)
- 缓存过期后仍可用(离线时),但会标记"缓存较旧"
- 有网络时会后台自动刷新

### 离线使用

断网时,已缓存的 URL 仍可打开(用缓存内容),顶部会提示离线状态。

---

## 书签

- 侧边栏顶部"📑 书签"区显示当前文档的书签
- 点击书签快速跳转到对应标题
- (书签在阅读过的标题处标记)

---

## 导出

- 点 📤 导出当前文档为 HTML
- 选择保存位置,生成带内联样式的 `.html` 文件
- 导出的 HTML 可独立打开,无需 MD Reader

---

## 夜间模式

- 点 🌙/☀️ 切换日间 / 夜间
- 默认跟随系统主题
- 手动切换后记住偏好,下次启动用你的选择

---

| 快捷键 | 功能 |
|--------|------|
| `⌘F` | 搜索 |
| `Esc` | 关闭搜索 |
| 拖拽 .md 到窗口 | 打开文件 |

## 常见问题

### Q: 打开提示"已损坏"
A: 见上文"安装"部分,终端跑 `xattr -cr "/Applications/MD Reader.app"`。

### Q: 网络 URL 打不开
A: 检查 URL 是否为 https(不支持 http),GitHub 链接格式是否正确。网络问题会用缓存(如果有)。

### Q: 文件夹里看不到某些文件
A: 只显示 `.md` / `.markdown` 文件;隐藏文件(以 `.` 开头)和空文件夹会被跳过。

### Q: 文件改了但没刷新
A: 确认保存了文件。监听有 500ms 防抖延迟(避免频繁刷新)。网络文档不会监听(没有本地文件)。

### Q: 换台 Mac 打不开
A: 当前只打了 Apple Silicon(aarch64)的包,Intel Mac 用不了。需要的话可以打 universal binary。

## 数据存储位置

应用数据存在系统的 Application Support 目录:
- 阅读进度:`progress.json`
- 最近打开:`recent.json`
- 书签:`bookmarks.json`
- 上次文件夹:`workspace.json`
- 网络缓存:`<hash>.json`

macOS 上路径通常是 `~/Library/Application Support/com.zcode.mdreader/`。
